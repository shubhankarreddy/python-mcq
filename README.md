# Python MCQ Skills Assessment

A practical Python multiple-choice quiz webpage for analyst, junior data science, and interview-ready Python skills.

Live site: https://shubhankarreddy.github.io/python-mcq/

## Contents

- P1 Basic: 25 questions
- P2 Intermediate: 50 questions
- P3 Advanced: 50 questions
- P2 + P3 make up 80% of the question bank
- Scenario-based questions with immediate feedback and explanations
- Topic breakdown after each level

## Levels

- P1 Basic: Python syntax, control flow, functions, collections, strings, file handling, exceptions, NumPy basics, and pandas basics
- P2 Intermediate: advanced pandas, missing values, GroupBy, merge/join/concat, pivot tables, vectorization, datetime operations, NumPy broadcasting, visualization, data cleaning, statistics, Flask, Streamlit, and EDA storytelling
- P3 Advanced: MultiIndex, rolling and expanding windows, transformation pipelines, performance tuning, advanced NumPy, memory optimization, simulations, large datasets, dashboard storytelling, project structure, logging, debugging, and production-ready code
